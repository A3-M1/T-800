# T-800

Repo GitHub pour le contrôle de notre super roomba aka T-800 

> Vous trouverez dans ce repo tous les scripts et autres fichiers que nous avons créé lors de l'UV LARM, que ce soit pour des produits fini ou juste des tests et autres tutos

## Installation

Pour installer les packages sur votre machine, **rendez vous dans votre ROS space** et **clonez ce repo** :

```console
git clone https://github.com/A3-M1/T-800.git
```

## Packages

Ce repo est composé de **plusieurs packages**, correspondants à plusieurs étapes ou besoin lors du projet.

> Un README est *(ou sera)* disponible pour chacun des packages, expliquant comment le build, comment ils fonctionne, etc... Il est situé dans le dossier du package.
